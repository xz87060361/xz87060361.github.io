创建博客流程

搭建步骤操作手册:

一.nodeJS安装与测试

二.安装hexo及创建本地博客

三.申请github帐号一枚(个人域名)

四.config.yml设置

五.发布web

六.测试

一.nodeJS安装与测试

1.首先打开以下网址进行下载

网址： https://nodejs.org/en/

2.安装完成后在终端输入:node-v进行测试

二.安装hexo及创建本地博客

1.安装hexo输入以下命令

2.安装有问题时(也可直接输入)

输入密码（提示：密码可能不显示但是可以输入）

3.查看软件安装进度上(测试)

4.创建博客文件夹并用终端状态进入该文件夹

5.打开博客服务器及博客地址

终端状态下关闭服务器命令：（Ctrl+c）

三.申请github帐号一枚

网址：https://github.com/

1.注册

2.输入码

3.邮箱验证信息

4.配置ssh-keygen(在终端上连续三次回车)

5.密钥

  • 输入（vim ~/.ssh/id_rsa.pub）完成回车进行并复制其内容
  • 反回原内容（”:q”）

6.粘贴到github中

7.进入终端本地操作与github验证

  • 输入git config –global user.name “登陆用户名”
  • 输入git config –global user.email “邮箱”

8.新建博客域名仓库（PS:用户名’.github.io’结束）

  • 打开文件夹进行config.yml找到deploy 设置如下：type: git
  • 并在type下面新建repo: 域名仓库的https地址
  • 注意（‘：’后要注意用空格隔开）

五.发布(可理解为上传到web让外网可访问)

在终端中输入以下空容：

  • npm install hexo-deployer-git–save
  • hexo g
  • hexo d
