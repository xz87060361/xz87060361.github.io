二、音视频

1.audio(音频)（属性）

  • controls (音频控制面板显示)
  • autoplay (自动播放)
  • loop （单曲循环）
  • preload (自动下载但不播放＊＊＊＊＊)
  • src （引入文件路径）
  • 可同source title共用兼容使用

2.video(视频)（属性）

  • controls (音频控制面板显示)
  • autoplay (自动播放)
  • loop （单曲循环）
  • preload (自动下载但不播放＊＊＊＊＊)
  • src （引入文件路径）
  • width （可设置宽度）
  • 可同source title共用兼容使用

3.source 兼容语句

  • title 属性
  • 单标签

4.track

  • 歌词文件
  • 字幕文件

5.embed

  • 引入flash
  • src 引入路径

三、短语句

1.mark 标记

  • 高亮文件
  • 常作用于搜索高亮

2.meter

  • 定义度量衡 （表示在一定范围内的值，如投票占比、使用量等）
  • value (实际度量的值，默认为0,可以为整数或者浮点数)
  • high （范围的上限值）
  • low （范围的下限值）
  • max （最大值，默认值是1）
  • min （最小值，默认值是0）
  • optimum(最值的值，必须在min属性值与max属性值之间)

3.progress

  • 定义进度
  • value
  • max
  • min

4.time

  • 定义时间和日期
  • 标签作用

5.wbr

  • 定义换行

6.引入短语句

  • cite 引入斜体不换行
  • q 带引号不换行
  • blockquote 换行并有缩进效果

7.code(格式化代码) pre

  • code只可写一行代码不换行
  • pre 可写多行代码，可换行

8.object param

  • object包含图像，音频，视频，falash使用
  • param配合object使用定义名字

9.abbr(缩写使用的标签)

10.bdi 定义隔离

四、表单元素

a.input元素

b.表单

1.label for

  • label点击之后自动选择文本框

2.datalist

  • 配合input使用
  • list 属性配合id名

3.ins del

  • ins 定义已经被插入文档中的文本：可以理解为下划线
  • del 定义删除：可以做用于商品打折

4.samp 转换(定义计算机程序的样本文本)

5.fieldset legend 可做标题

五、表单属性

  • 1.autofocus 自动默认在一个文本框内，autofocus 属性规定当页面加载时按钮应当自动地获得焦点
  • 2.placeholder 提示框中输入浅色信息
  • 3.required 强行规定输入@格式用于email
  • 4.pattern 填写正确格式内容
  • 5.autocomplete 提交时提示填写完整内容
  • 6.from 用于表单框结构
