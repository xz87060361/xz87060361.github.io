博客搭建

一.安装hexo

1.安装hexo

2.安装有问题时

输入密码（提示：密码可能不显示但是可以输入）

3.查看软件安装进度上(测试)

4.创建目录并进入

5.打开博客服务器及博客地址

关闭服务（Ctrl+c）

二.github帐号创建流程

a.网址：https://github.com/
b.注册

1.注册

2.输入码

3.验证信息

4.配置ssh-keygen(在终端上连续三次回车)

5.密钥

a.输入完成回车进行并复制其内容

b.反回原内容（”:q”）

6.粘贴到github中

7.进入终端本地操作与github验证

a.输入git config –global user.name “登陆用户名”

b.输入git config –global user.email “邮箱”

8.新建博客域名仓库（PS:用户名’.github.io’结束）

三.博客config.yml(ps:保存)

a.设置deploy中的 type: git

b.新建repo: 域名仓库的https地址


四.发布

1.npm install hexo-deployer-git–save

2.hexo g

3.hexo d

五.测试
